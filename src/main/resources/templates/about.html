Acerca de FlightApp

FlightApp es una aplicación web para simular el proceso de compra de un boleto de avión.

El objetivo principal de este proyecto es mostrar un flujo de trabajo, desde que el usuario completa un formulario con los datos de su vuelo hasta la generación y descarga de un boleto en formato PDF.

En la construcción de esta APP, se ponen en práctica tecnologías como:

  • Spring Boot: Para el backend y la lógica de negocio.
  • Thymeleaf: Para la renderización dinámica de las páginas HTML.
  • JPA/Hibernate: Para la persistencia de datos en una base de datos PostgreSQL.
  • OpenPDF: Para la generación de documentos PDF al momento.

Proyecto sujeto a implementación de nuevas funcionalidades de manera gradual.
